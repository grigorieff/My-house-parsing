# My-house-parsing.
This repository contents package, used for www.reformagkh.ru parsing.

## Installation:

You can copy this module using the following line:

```bash
pip install https://github.com/alexeykosh/My-house-parsing
```

## Examples of usage:

First of all, if you want to import it in your Python project, you need to write the following:

```Python
from myhouse import get_table
```

Then, for example, you need to have passport tables from [this url](https://www.reformagkh.ru/myhouse/profile/view/7561322). So, you write the following line of code:

```Python
get_table(url='https://www.reformagkh.ru/myhouse/profile/view/7561322')
```
